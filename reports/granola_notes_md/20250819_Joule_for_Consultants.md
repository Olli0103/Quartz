### Joule for Consultants Overview

- New AI tool rolled out to all SAP consulting colleagues in July/August
- Currently 9,000 active users across ASC, GCNS, and PE
- Mandatory usage in all SAP-delivered engagements starting September 1st
- Provides information retrieval from SAP’s proprietary and public knowledge base
  - Troubleshooting support
  - Clean core concept explanations
  - ABAP code analysis and generation
  - Step-by-step guidance

### Current Capabilities & Limitations

- Does not access customer systems or perform system actions (yet)
- Limited to knowledge-based assistance and code analysis
- Built on separate technical stack from standard Joule (20% integration)
- Full harmonization with standard Joule stack planned by end of year

### Integration & Roadmap

- System integration capabilities coming end of 2024
- Potential ServiceNow integration for automatic prompt creation
- Official internal and external roadmaps available (to be shared)
- Product team actively accepting feedback for improvements

### CAS-Specific Rollout Plans
