### J4C Product Roadmap & Current Features

- Content expansion from 9GB to 12GB (end 2025 to present)
  - Added SAP notes and additional documentation
  - Custom pilot launched late 2025, GA planned May-June 2026
- Prompt library enhancements
  - Customer-managed libraries at organization level
  - Admins can add custom prompts for their companies
  - “Try Joule” button launches prompts directly in Joule instance
- File upload capabilities
  - Current: PDF, TXT, images
  - Next release: DOCX files
  - Testing additional file types on request
- UX improvements
  - Copy function now generates markdown with URL citations (not bracket references)
  - Web search available via admin toggle

### Expert Workspace Feature (Upcoming)

- Addresses 7-day conversation expiration limitation
- Users can create named experts with persistent context
  - Custom instructions and descriptions
  - File uploads maintained across sessions
  - Project-specific configurations
- Eliminates need to re-enter long prompts repeatedly
- Designed for project-specific workspaces with relevant documentation

### Future Roadmap Priorities

- Landscape awareness (major 2026 investment)
  - Learn customer software versions and configurations
  - Enable tailored J4C responses based on user landscape
- Agent development with connectors
  - Early beta agents to be announced at Sapphire
- Knowledge base expansion and partner extensibility
- AI-driven migration tools
  - Transition guide for S/4 migration journeys
  - Public and private cloud support

### Q&A Key Points

- Security model same as Joule (data not stored, no LLM training on uploaded files)
- No direct ChatGPT integration planned
- Code generation not supported (explains code only)
- Migration agent separate from J4C for ECC to S/4 conversion
- PowerPoint files not currently supported in expert workspace
- 7-day conversation limit due to data privacy policies across regions
