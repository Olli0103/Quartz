### Key Dashboard Features & Functionality

- Standard filtering capabilities
  - SKU, date, region filters pre-configured
  - Custom filters available via plus symbol
  - Bookmark feature saves filter selections as defaults
- Data export functionality
  - Multiple file formats available
  - Custom naming and configuration options
- Revenue Analysis dashboard
  - Shows booked revenue (not ACV) by year
  - 2024: €10M, 2025: €11.3M
  - Year-over-year comparison with monthly granularity
  - Last data refresh: February 21, 2026
- Contract Management dashboard
  - Three pages: new engagement, delivery readiness, end date monitoring
  - Tracks ASM assignments and contract readiness
  - Identified Sony Par missing ASM assignment (starts October)
- Pipeline & Booking report available but currency-only view
  - Missing package count data (still requires Cloud Reporting)
- Consumption report limited to volume-based services
  - Only covers ticket buckets, EMS
  - Currently only Fraunhofer customer data
