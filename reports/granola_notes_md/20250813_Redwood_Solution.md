### Redwood Run My Jobs Cloud Overview

- Current job scheduling tools available: UC4, Control M, and Redwood Run My Jobs Cloud
- Redwood Run My Jobs Cloud (RMJ) is SaaS successor to legacy Business Process Automation (BPA)
  - BPA was on-premise tool, no longer used
  - RMJ runs in Redwood’s cloud environment
- Only RMJ gets managed VM support from ECS
  - Other tools require customer to buy and manage Infrastructure as a Service
  - ECS installs and maintains communication agent between private cloud and SaaS
  - Handles OS updates and VM management completely

### Service Offering - Advanced Job Management

- Part of Advanced Monitoring service portfolio under Manish’s ownership
- Two main service areas:
  - Service request fulfillment
    - Job termination, scheduling, rescheduling
    - New job chain development
    - Basic job management tasks
  - Monitoring and remediation
    - 24/7 monitoring of all jobs and job chains
    - Technical issue remediation and rescheduling
    - Escalation to Redwood for complex technical issues
    - Customer involvement for application-level problems
- Excludes: New job development requiring business knowledge (remains with customer application teams)

### Cross-System Job Orchestration

- Enables process chains across multiple systems
- Example: Job in private cloud S/4 system triggers job in public cloud system
- Provides end-to-end workflow automation across hybrid environments
