### WalkMe Embedded Features Assessment

- One-click opt-in activation available for all SAP SuccessFactors customers
  - User interface-based, takes \~2 seconds to activate
  - Cannot showcase as CAS value-add since customers can do independently
- 80%+ opt-in rate reported from SuccessFactors (June data)
- Features include:
  - In-app guidance and click-through assistance
  - Hover tooltips and navigation hints
  - Step-by-step wizards for form completion
  - Language switching support
- No WalkMe Insights available for embedded version
  - Reporting only accessible with premium/platform subscriptions
  - Would create license dependency for any CAS involvement

### Service Overlap Concerns

- WalkMe already provides comprehensive service coverage:
  - Auto-attached premium SKU for activation services
  - Generic freestyle SKU for platform customers


  - Covers orientation, reporting, feature training
  - Fully managed by WalkMe consulting teams
- CAS entry would duplicate existing services
  - Customers already have dedicated WalkMe CSMs
  - Commercial overlap with established engagement model

### CAS Capability Limitations

- Only 2 certified WalkMe resources in CAS organization
  - Oliver and Yatti as sole certified consultants
  - Scalability concerns for any service expansion
- Customer dependency requirements:
  - Must provide WalkMe ID for any ticket-based work
  - License prerequisites for configuration changes
  - Limited to building solutions for end users, not consultants

### Pilot Feasibility Analysis

- US payroll pilot concept discussed but not pursued
  - Resource constraints prevented implementation
  - Would require separate pilot structure outside current operations
- Potential ticket-based support model:
  - Could consume operate ticket bucket
  - Risk of stepping on WalkMe partner toes
  - Customer benefit: single contract vs multiple SKUs

### Strategic Recommendation

- No viable CAS service model identified
  - Insufficient value-add over existing WalkMe offerings
  - Resource scalability challenges
  - Commercial conflict with partner engagement model
- Focus remains on other CAS priorities given limited capacity
- WalkMe Insights monitoring could support solution adoption tracking
  - Would require coordination with solution CSMs
  - Secondary priority given current workload
