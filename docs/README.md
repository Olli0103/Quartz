# Quartz Notes Documentation

Welcome to the official documentation for **Quartz Notes** — a premium native markdown notes app for Apple platforms.

Quartz Notes stores your notes as plain Markdown files in folders you control. No proprietary database, no cloud account required, no vendor lock-in. Your notes are always portable and always yours.
