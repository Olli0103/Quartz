# Dashboard Overview

The Dashboard is your command center — the first thing you see when you open Quartz Notes. It provides an at-a-glance view of your vault and quick access to common actions.

## Accessing the Dashboard

- Click **Dashboard** in the sidebar
- The Dashboard appears when no note is selected
- On launch, the Dashboard is shown by default (configurable in Settings)

## Sections

### Greeting & Quick Actions

The top row shows a time-based greeting ("Good Morning", "Good Afternoon", "Good Evening") with the current date. Quick action buttons on the right provide one-click access to:

- **New Note** — Create a note immediately
- **Graph** — Open the knowledge graph
- **Voice** — Record a voice note
- **Meeting** — Start meeting minutes

### Vault Statistics

Three key numbers displayed prominently:

- **Notes** — Total markdown files in the vault
- **Folders** — Total directories
- **Open Tasks** — Unchecked `- [ ]` items across recent notes

### Quick Capture

A text field for capturing fleeting thoughts. Type a thought and press **Enter** to append it to today's Daily Note with a timestamp. The Daily Note is created automatically if it doesn't exist.

### AI Briefing

An AI-generated summary of your most recently edited notes. Requires an AI provider configured in Settings. The briefing is cached for 4 hours per vault.

### Serendipity

Surfaces a random note from your vault — or a note from exactly one year ago ("on this day"). Click the note to open it, or click **Shuffle** to pick another.

### Recent Notes & Action Items

Two side-by-side panels:

- **Recent Notes** — The 8 most recently modified notes with title and relative date
- **Action Items** — Open `- [ ]` tasks parsed from your 15 most recent notes. Click the circle to mark a task complete (writes the change back to the source file).

### Activity Heatmap (Momentum)

A GitHub-style contribution graph showing your writing activity over the past 26 weeks. Each cell represents one day, colored by the number of notes you edited that day. Hover over any cell to see the exact date and edit count.
