# Appearance Settings

Customize the visual appearance of Quartz Notes. Open via **Quartz Notes > Settings > Appearance** (or Cmd+,).

## Theme

Choose between three modes:

| Theme | Description |
|-------|-------------|
| **System** | Follows your macOS dark/light mode setting |
| **Light** | Always light background |
| **Dark** | Always dark background |

## Accent Color

Select from 7 preset accent colors: Blue, Red, Green, Orange, Purple, Pink, Gray. The accent color is used for selection highlights, buttons, tags, and the activity heatmap.

## Editor Font Size

Adjust the editor font size from **12pt to 24pt** using the slider. A live preview shows how text will look at the selected size. This affects the editor, syntax highlighting, and heading sizes.

## Vibrant Transparency

When enabled, the sidebar and note list use macOS system vibrancy (frosted glass materials). When disabled, they use a solid opaque background. Disable this if you find the translucency distracting.

## Pure Dark Mode

When enabled (and using Dark theme), the app uses true black (#000000) backgrounds instead of dark gray. Ideal for OLED displays and users who prefer maximum contrast in dark mode.

Applies to all three columns: sidebar, note list, and editor.

## Show Dashboard

Toggle whether the Dashboard appears when no note is selected. When disabled, an empty state is shown instead.
