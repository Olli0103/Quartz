# Editor Settings

Configure the writing experience. Open via **Quartz Notes > Settings > Editor**.

## Writing

| Setting | Description |
|---------|-------------|
| **Focus Mode** | Hides all UI chrome for distraction-free writing |
| **Typewriter Mode** | Keeps the active line vertically centered |

## Behavior

| Setting | Default | Description |
|---------|---------|-------------|
| **Autosave** | On | Automatically saves 1 second after you stop typing |
| **Spell Check** | On | macOS spell checking in the editor |

## Typography

### Font Family

Choose from four font families:

| Family | System Font | Best for |
|--------|------------|----------|
| **System** | SF Pro | General writing (default) |
| **Serif** | New York | Long-form prose, articles |
| **Monospaced** | SF Mono | Code-heavy notes, technical writing |
| **Rounded** | SF Rounded | Casual, friendly aesthetic |

### Font Size

Adjust from **12pt to 24pt**. Affects body text; headings scale proportionally (H1 = 1.7x, H2 = 1.45x, H3 = 1.25x, etc.).

### Line Spacing

Adjust from **1.0x to 2.5x**. Higher values add more space between lines. Default is 1.5x.

### Text Width

Adjust from **400pt to 1200pt**. Controls the maximum width of text in the editor. On wide screens, text is centered within this width with generous margins. Default is 720pt.

### Live Preview

The settings panel includes a live preview paragraph that updates in real time as you adjust font family, size, and line spacing.
