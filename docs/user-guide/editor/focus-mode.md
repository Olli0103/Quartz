# Focus Mode

Focus Mode hides all UI elements except the editor, giving you a distraction-free writing environment.

## Activating Focus Mode

- Click the **expand arrows** button in the toolbar
- Or toggle it in **Settings > Editor > Focus Mode**

When active, the sidebar, note list, toolbar, and status bar fade away. Only the text editor remains.

## Exiting Focus Mode

- Click the **collapse arrows** button (still visible in the toolbar area)
- Press the same keyboard shortcut again

## Typewriter Mode

Typewriter Mode keeps the active line vertically centered in the editor. Enable it in **Settings > Editor > Typewriter Mode**.

When combined with Focus Mode, this creates a calm writing experience where your current sentence is always in the center of the screen.

## Accessibility

Focus Mode respects the **Reduce Motion** accessibility setting. When Reduce Motion is enabled, the transition between normal and focus mode uses a simple crossfade instead of the animated slide.
