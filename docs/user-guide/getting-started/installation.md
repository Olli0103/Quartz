# Installation

## System Requirements

- **macOS 15 (Sequoia)** or later
- Apple Silicon (M1 or later) or Intel Mac
- 100 MB free disk space (plus space for your notes)

## Installing from TestFlight

1. Open the TestFlight invitation link you received
2. Click **Accept** in TestFlight
3. Click **Install** to download Quartz Notes
4. Open Quartz Notes from your Applications folder or Launchpad

## Installing from the Mac App Store

> Coming soon. Quartz Notes is currently in beta via TestFlight.

## Building from Source

If you have Xcode 16 or later installed:

1. Clone the repository: `git clone https://github.com/Olli0103/Quartz.git`
2. Open `Quartz.xcodeproj` in Xcode
3. Select the **Quartz** scheme and your Mac as the destination
4. Press **Cmd+R** to build and run

## First Launch

When you open Quartz Notes for the first time, the onboarding wizard will guide you through:

1. **Welcome screen** — overview of what Quartz Notes does
2. **Choose a vault folder** — select an existing folder or create a new one
3. **Choose a structure** — pick PARA, Zettelkasten, or start with an empty vault

After onboarding, Quartz Notes remembers your vault across launches using a secure bookmark.
