# The Interface

Quartz Notes uses a three-column layout inspired by Apple Mail and Apple Notes.

## Three-Column Layout

| Column | Name | Purpose |
|--------|------|---------|
| **Left** | Sidebar | File tree, folders, tags, quick access, dashboard |
| **Middle** | Note List | Filtered list of notes with previews |
| **Right** | Detail | Editor (when a note is selected) or Dashboard |

## Sidebar (Left Column)

The sidebar contains:

- **Dashboard** — Click to return to the command center
- **Quick Access** — All Notes, Favorites, Recent
- **Tags** — Horizontal scrolling tag filter
- **File Tree** — Your vault's folder structure with drag-and-drop
- **Map View** — Link to the knowledge graph
- **Trash** — Access deleted notes
- **Switch Vault** — Open a different vault

### Sidebar Actions

- **New Note** — Click the "+" button or press Cmd+N
- **New Folder** — Available in the ellipsis menu
- **Search** — Type in the floating search bar at the bottom of the sidebar

## Note List (Middle Column)

Shows notes matching the current sidebar selection:

- **Title** — Bold, extracted from frontmatter or first heading
- **Timestamp** — Relative date ("2 min ago", "Yesterday")
- **Snippet** — First 2-3 lines of the note body
- **Tags** — Color-coded tag chips

### Sorting

Click the sort icon in the toolbar to sort by:
- Date Modified (newest first)
- Date Created
- Title (A-Z)

## Detail Pane (Right Column)

Displays either:

- **Dashboard** — When no note is selected (or when you click "Dashboard" in the sidebar)
- **Editor** — When you select a note from the note list or sidebar

## Toolbar

The macOS toolbar contains:

- **Formatting buttons** — Bold, Italic, Headings, Lists, Code, Links
- **Voice Note** — Record audio to create a note
- **Share** — Export the current note
- **Focus Mode** — Toggle distraction-free writing
- **Chat** — AI chat about the current note
- **Inspector** — Toggle the right-side inspector panel

## Inspector Panel

The inspector (right side, toggled via the sidebar icon in the toolbar) shows:

- **Table of Contents** — Clickable heading outline
- **Document Stats** — Word count, character count, reading time
- **Frontmatter** — Tags, creation date, custom metadata
