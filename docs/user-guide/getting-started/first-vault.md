# Creating Your First Vault

A **vault** is simply a folder on your Mac that contains your notes. Every note is a standard `.md` (Markdown) file. You can have one vault or many, and switch between them at any time.

## What is a vault?

- A regular folder on your file system
- Contains `.md` files (your notes) organized in subfolders
- No proprietary database — you can open your notes in any text editor
- Can live anywhere: local disk, iCloud Drive, Dropbox, or an external drive

## Creating a new vault

1. On first launch, click **Get Started** on the welcome screen
2. Click **Choose Folder**
3. In the file picker, navigate to where you want your vault
4. Click **New Folder** to create a dedicated folder (e.g., "My Notes")
5. Select the folder and click **Choose**

## Choosing a vault structure

After selecting a folder, Quartz Notes offers three starter templates:

| Template | What it creates | Best for |
|----------|----------------|----------|
| **PARA Method** | Projects, Areas, Resources, Archive folders | Productivity-focused note-taking |
| **Zettelkasten** | Fleeting, Literature, Permanent Notes folders | Academic research and connected thinking |
| **Empty Vault** | Nothing — start with a blank canvas | Users who prefer their own organization |

You can always reorganize folders later. The template just gives you a head start.

## Opening an existing vault

If you already have a folder of Markdown notes:

1. Go to **File > Open Vault** (Cmd+Shift+O)
2. Select your existing notes folder
3. Quartz Notes will scan and index all `.md` files automatically

## Switching between vaults

- **From the sidebar:** Click **Switch Vault** at the bottom of the sidebar
- **From the menu:** File > Open Vault (Cmd+Shift+O)
- **Create a new vault:** File > Create New Vault

Quartz Notes remembers the last vault you opened and restores it on launch.
