# Design Review: $ARGUMENTS

## Protocol

1. **Spawn apple-design-specialist agent** to review:
   - Human Interface Guidelines compliance
   - Platform conventions (iOS/iPadOS/macOS/visionOS)
   - Visual consistency
   - Typography and spacing
   - Material usage
   - Animation appropriateness

2. **Apple Design Award criteria check**:
   - [ ] Innovation: Novel approach?
   - [ ] Delight: Moments of pleasure?
   - [ ] Interaction: Intuitive, natural?
   - [ ] Visuals: Beautiful, consistent?
   - [ ] Inclusivity: Works for everyone?

3. **Platform fidelity check**:
   - Does it feel native?
   - Would Apple build it this way?
   - Is it consistent with system apps?
   - Does it respect platform conventions?

4. **Generate report**:
   - HIG violations
   - Design recommendations
   - Specific improvements with rationale
   - References to HIG sections

## Output

Return design review with:
- Current state assessment
- Specific violations
- Concrete improvements
- HIG/WWDC references
